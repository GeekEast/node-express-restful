
## Flow
- Install `brew install mongodb`
- Local DB 
```sh
sudo mkdir -p /data/db
sudo chown -R `id -un` /data/db
```
- Run `mongod` or `mongod --config /usr/local/etc/mongod.conf`
- Run by brew `brew services start/stop mongodb`
